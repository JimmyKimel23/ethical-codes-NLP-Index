# Competitor Relations Ethics Codes Corpus

Стартовый репозиторий для анализа корпоративных кодексов этики по теме **«Отношения с конкурентами»**.

## Что внутри

- `materials/raw/` — исходные файлы кодексов.
- `materials/processed/` — текстовые версии, извлечённые из PDF/HTML/DOCX при необходимости.
- `registry/codes_index.csv` — инвентаризация корпуса.
- `registry/codes_status.csv` — сводный статус анализа по документам.
- `analysis/yaml/` — YAML-карточки по каждому кодексу.
- `synthesis/` — файлы для синтеза идеального раздела.
- `docs/` — методология, шкалы статусов, реестр принципов.
- `prompts/` — модульные инструкции для Codex.

## Минимальный рабочий цикл

1. Добавить кодексы в `materials/raw/`.
2. Обновить `registry/codes_index.csv`.
3. Проанализировать каждый кодекс и сохранить YAML-карточку в `analysis/yaml/`.
4. После каждого анализа обновить `registry/codes_status.csv`.
5. После накопления достаточного корпуса выполнить синтез в `synthesis/`.

## Базовые правила

- Анализ ведётся по языку оригинала.
- Статусы, нормализация и аналитические выводы фиксируются на русском.
- Каждое существенное утверждение должно опираться на текст документа.
- Синтез не делается «из головы»: только на основе завершённых карточек.
