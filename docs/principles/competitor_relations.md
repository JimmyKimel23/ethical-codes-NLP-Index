# Канонические принципы — competitor_relations

## Основные коды

- `COMP01` — Не пытаться узнать конфиденциальную информацию конкурента 
- `COMP02` — Не переманивать ключевых работников конкурента с целью ослабить его позиции
- `COMP03` — Не вступать в сговор с конкурентами о ценах, объемах или разделе рынка
- `COMP04` — Сотрудничать с конкурентами с целью повышения эффективности рынка 
- `COMP05` — Поддерживать уважение к конкурентам и подчеркивать важность деятельности
- `COMP06` — Не пытаться ослабить конкурентов путем ограничения их доступа к каким-либо ресурсам 
- `COMP07` — Соблюдать права интеллектуальные собственности конкурентов
- `COMP08` — Не распространять ложную и порочащую информацию о конкурентах
- `COMP09` — Предостерегать конкурентов от нарушения данных принципов с их стороны. 
- `COMP10` — Сообщать регулятору о замеченных неэтичных действиях конкурентов.
- `COMP99` — иной принцип, не покрытый основным набором.

## Правила использования

1. Сначала соотноси фрагмент с `COMP01-COMP10`.
2. `COMP99` используй только если не подходит ни один канонический код.
3. Для `COMP99` обязательно заполняй:
   - `other_principle_label`
   - `other_principle_explanation`
